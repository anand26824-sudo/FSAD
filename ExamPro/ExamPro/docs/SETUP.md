# Setup Guide

## Requirements
- Modern browser (Chrome, Firefox, Edge, Safari)
- Optional: Node.js for live-server

## Installation
```bash
# Clone the project
git clone https://github.com/yourorg/exampro.git
cd exampro

# Install dev dependency
npm install

# Start development server
npm start
```

## Configuration
Edit `config/config.js` to change:
- Pass mark threshold
- Grade boundaries
- API base URL (for backend integration)
- Session timeout
