# ExamPro — Dynamic Online Examination & Result Processing System

## Overview
ExamPro is a fully client-side online examination system built with vanilla HTML, CSS, and JavaScript.

## Features
- Student login & session management
- Live exam dashboard with status indicators
- Timed MCQ examination engine
- Auto-grading & result processing
- Detailed answer review
- Admin panel for managing exams & students

## Project Structure
```
ExamPro/
├── index.html          # Entry point
├── assets/             # CSS, JS, Images
├── pages/              # HTML pages
├── components/         # Reusable UI components
├── data/               # JSON data files
├── admin/              # Admin panel
├── student/            # Student portal
├── config/             # App configuration
├── utils/              # Helper utilities
└── docs/               # Documentation
```

## Getting Started
1. Clone or download this project
2. Open `index.html` in a browser
3. Or run: `npx live-server .`
